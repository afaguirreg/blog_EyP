IOT2000

La familia SIMATIC IOT2000 (2020/2040/2050) es una plataforma abierta para recopilar, procesar y transferir datos en el entorno de producción. Ideal como puerta de enlace de red de datos entre el nivel de Intranet, la nube o TI y las producciones. El sistema abierto de este dispositivo admite numerosos protocolos de comunicación y programación tanto en lenguaje de programación de alto nivel como en lenguaje de programación de bajo nivel para permitir soluciones personalizadas.

La familia SIMATIC IOT es compatible con Linux (YOKTO) y soluciones de comunicación flexibles en protocolos industriales populares como MODBUS, RTU, OPC UA, FROFINET, REST, MQTT y AMQP. Este dispositivo también es compatible con la mayoría de los códigos Arduino que se pueden cargar directamente desde el IDE. Otros programas también se pueden compilar y ejecutar bajo YOCTO Linux en el controlador usado X1000 SOC (X86). Otros dispositivos de hardware como PLC, los sensores que usan Ethernet se pueden integrar a través de (ETH). ) Ethernet o serie a través del adaptador USB utilizando muchos protocolos disponibles como mencionamos anteriormente.

Hay muchos lenguajes de programación compatibles con SIMATIC IOT, como Java, C/C++, Python a través de una gama de IDE que incluye Eclipse y los compiladores para YOKTO Linux. Estos dispositivos también se pueden expandir a través del puerto mPCI integrado. Node-RED, MQTT y SQLITE también se pueden usar para conectar y controlar dispositivos periféricos locales y remotos. A continuación se muestran los pasos de cómo vamos a integrar Arduino IDE para comunicarnos y programar con SIMATIC IOT 2000/2020/2040.


Los dispositivos de la familia IOT2000 disponen de conectores para acceder a sus entradas y salidas.

Se puede utilizar un módulo SIMATIC I/O Shield o un dispositivo con factor de forma para Arduino.



Configuración IOT2000: Elementos necesarios

- IOT 2020/2040/2050

- Fuente de voltaje 9 36V DC

- Tarjeta de memoria Micro SD (Min 8GB)

- Cable Micro USB

Instalar el sistema operativo en la memoria Micro SD

1. Descargar e instalar Balena Etcher https://www.balena.io/etcher/

2. Descargar el archivo de imagen del sistema operativo para el dispositivo (Requiere inicio de sesión en https://support.industry.siemens.com/):

IOT2040: https://support.industry.siemens.com/cs/attachments/109741799/IOT2040_Example_Image_V3.1.1.zip

IOT2050: https://support.industry.siemens.com/cs/attachments/109741799/IOT2050_Example_Image_V1.1.1.zip

3. Insertar la tarjeta de memoria Micro SD en la ranura del lector del PC.

4. Escribir el sistema operativo en el dispositivo utilizando Balena etcher

5. Instalar la tarjeta de memoria Micro SD en el dispositivo IOT2000
