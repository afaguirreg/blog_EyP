3.5 LCD Assistant



LCD Assistant es un programa portable que permite convertir imagenes monocromáticas en código para utilizar con pantallas compatibles.

Descarga de LCD Assistant:

https://en.radzio.dxp.pl/bitmap_converter/LCDAssistant.zip

Buscar y descargar la imagen deseada


Abrir la imagen con un editor (MS Paint)


Guardar la imagen en formato .bmp (Mapa de bits monocromático)


Abrir LCD Assistant y seleccionar el menu File --> Load image


Seleccionar el archivo de imagen en formato .bmp


Realizar las configuraciones necesarias y seleccionar el menu File --> Save output


Guardar con extensión .h en la carpeta del proyecto (archivo .ino)


Editar el archivo con extensión .h (Tipo de dato del vector)
