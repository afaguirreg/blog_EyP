Sistema para medición de humedad y temperatura

Descripción: El módulo de instrumentación consiste en una caja metálica ensamblada con display, teclado e interfaces para comunicación y transductores de humedad y temperatura y podrá ser usada para medir y registrar el valor de éstas variables físicas a través de un puerto de comunicaciones.

Características:

- Carcasa metálica, display gráfico, comunicación RS-232
