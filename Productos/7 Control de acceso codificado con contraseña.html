Control de acceso codificado con contraseña

Descripción: El control de acceso consiste en un módulo instalado a la entrada de un área donde se requiere identificar el personal que ingresa o sale a determinada hora del día.

Características:

- Utiliza tags tipo tarjeta y tipo llavero para identificación de usuario

- Carcasa metálica, display gráfico, comunicación inalámbrica con PC
